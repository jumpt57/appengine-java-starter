# appengine-java-starter

Starter for appengine with :

* Java
* Maven
* Guice
* Java Bean validation
* Endpoint Framework
* Junit
* Testng
* Mockito
* BaseDevAppServerTestConfig (e2e testing)

Start with :
```
mvn clean appengine:run
```

Access API Explorer with :
```
http://localhost:8080/_ah/api/explorer
```
